InputEvents v1.4.0
An easy to use but comprehensive Event Library for Buttons, Encoders, Encoder Buttons, Analog Inputs, Joysticks and Switches.
README

This directory is for the PinAdapter and DebounceAdapter base/interfaces and concrete implementations.

There are currently implementations for regular GPIO pins, virtual pins (for programatically setting an apapter oin state) and GPIO Expander pins.

PinAdapters can be mixed with PinMixerAdapter and debounced with an implementation of a DebounceAdapter

The default debouncer FoltmanDebounceAdapter is used by default (but optional) when InputEvents buttons and switches are created, so do not have to be explicitly created.
